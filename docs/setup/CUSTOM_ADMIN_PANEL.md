# Кастомная админ-панель LootLink

## Обзор

Полнофункциональная кастомная админ-панель с современным дизайном для управления маркетплейсом LootLink.

### Особенности:

- **Современный дизайн** - Темная боковая панель, светлый контент
- **Полный функционал** - Все что нужно для управления маркетплейсом
- **AJAX операции** - Быстрые действия без перезагрузки страницы
- **Графики и аналитика** - Chart.js для визуализации
- **Права доступа** - Разделение для админов и модераторов
- **Адаптивный дизайн** - Работает на мобильных устройствах

---

## Доступ

### URL:
```
http://ваш-сайт/custom-admin/
```

- **Владелец** (`is_superuser`) - Полный доступ ко всему
- **Администратор** (`is_staff`) - Полный доступ к управлению

## 🔌 API Endpoints

Все API endpoints доступны только для авторизованных админов/модераторов:

### Пользователи:
```
POST /custom-admin/api/users/<id>/verify/     - Верифицировать
POST /custom-admin/api/users/<id>/ban/        - Заблокировать
POST /custom-admin/api/users/<id>/unban/      - Разблокировать
POST /custom-admin/api/users/<id>/moderator/  - Назначить модератором
```

### Объявления:
```
POST /custom-admin/api/listings/<id>/approve/  - Одобрить
POST /custom-admin/api/listings/<id>/reject/   - Отклонить
POST /custom-admin/api/listings/<id>/delete/   - Удалить
```

### Сделки:
```
POST /custom-admin/api/transactions/<id>/cancel/  - Отменить
```

### Споры:
```
POST /custom-admin/api/disputes/<id>/resolve/  - Разрешить
```

### Жалобы:
```
POST /custom-admin/api/reports/<id>/process/  - Обработать
```

### Статистика:
```
GET /custom-admin/api/stats/?period=7  - Получить статистику
```

## 💻 Использование AJAX

### Пример вызова API:

```javascript
// Верификация пользователя
async function verifyUser(userId) {
    const result = await adminAjax(
        `/custom-admin/api/users/${userId}/verify/`
    );
    
    if (result.success) {
        showToast(result.message, 'success');
        location.reload();
    } else {
        showToast(result.message, 'danger');
    }
}
```

### Глобальные функции:

```javascript
// AJAX запрос
window.adminAjax(url, data)

// Toast уведомление
window.showToast(message, type)
```

## Настройка и кастомизация

### Изменение цветов:

В `admin_panel/templates/admin_panel/base.html` найти раздел `:root` и изменить переменные:

```css
:root {
    --admin-primary: #6366F1;    /* Основной цвет */
    --admin-secondary: #EC4899;  /* Вторичный */
    --sidebar-bg: #1F2937;       /* Цвет sidebar */
}
```

### Добавление новых разделов:

1. Создать view в `admin_panel/views.py`
2. Добавить route в `admin_panel/urls.py`
3. Создать template
4. Добавить ссылку в sidebar (`base.html`)

### Добавление API endpoint:

1. Создать функцию в `admin_panel/api_views.py`
2. Добавить route в `urls.py` (раздел API)
3. Добавить JS функцию в template

## Решение проблем

### Счетчики в sidebar не обновляются

**Проблема**: В sidebar показываются нули

**Решение**: Убедитесь что middleware добавлен в `MIDDLEWARE` в `settings.py`:

```python
MIDDLEWARE = [
    # ... другие middleware
    'admin_panel.middleware.AdminPanelContextMiddleware',
]
```

### 404 ошибка при открытии админки

**Проблема**: Страница не найдена

**Решение**: Проверьте что URL добавлен в `config/urls.py`:

```python
urlpatterns = [
    path('custom-admin/', include('admin_panel.urls')),
    # ...
]
```

### AJAX запросы не работают

**Проблема**: Ошибка 403 CSRF

**Решение**: Проверьте что CSRF токен передается в запросах. Функция `adminAjax()` уже включает это.

## 📚 FAQ

### Q: Как дать кому-то доступ к админ-панели?

**A**: Через Django Admin:
1. Открыть `/admin/`
2. Accounts → Profiles
3. Найти пользователя
4. Поставить галочку "Is moderator"

Или назначить staff:
1. Accounts → Custom users
2. Найти пользователя
3. Поставить "Staff status"

**A**: 
- Красивый современный дизайн
