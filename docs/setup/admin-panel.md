# Кастомная админ-панель - Быстрый старт

## Что уже сделано:

### Создана структура приложения `admin_panel`
- Views для всех разделов
- API endpoints для AJAX операций
- Templates с современным дизайном

- Темная боковая панель (sidebar)
- Современные карточки и таблицы
- Графики Chart.js
- AJAX операции
- Toast уведомления

### Функционал:
- Dashboard с аналитикой
- Управление пользователями

## Быстрый запуск

### 1. Локальное тестирование:

```bash
# Применить миграции (если нужно)
python manage.py migrate

# Запустить сервер
python manage.py runserver

# Открыть в браузере
http://127.0.0.1:8000/custom-admin/
```

### 2. Деплой на продакшн:

```bash
# На локальном компьютере
git add .
git commit -m "feat: добавлена кастомная админ-панель"
git push origin main

# На сервере
ssh root@91.218.245.178
cd /opt/lootlink
git pull origin main
source venv/bin/activate
python manage.py migrate
python manage.py collectstatic --noinput
sudo systemctl restart lootlink
```

### 3. Первый вход:

```
URL: http://ваш-сайт/custom-admin/

Логин: ваш admin логин
Пароль: ваш admin пароль
```

## Полезные советы

### Как модерировать объявления:

1. Зайти в `/custom-admin/`
2. Если есть необработанные - кнопка "Перейти к модерации"
3. Просмотреть карточки с превью
4. Одобрить или отклонить

### Как разрешить спор:

1. Открыть `/custom-admin/disputes/`
2. Выбрать активный спор
3. Прочитать информацию от обеих сторон
4. Нажать "Разрешить"
5. Выбрать победителя и написать решение

### Как заблокировать пользователя:

1. Открыть `/custom-admin/users/`
2. Найти пользователя
3. Нажать иконку блокировки
4. Подтвердить действие

## Преимущества нашей реализации

### По сравнению с готовыми решениями:

#### Vs Django Admin:
- Красивый дизайн
- Визуальная модерация
- Графики и аналитика
- Лучший UX

#### Vs Jazzmin/Grappelli:
- Полностью под твой проект
- Кастомные функции
- Нет лишнего
- Легко расширять

#### Vs полностью кастомная React админка:
- Быстрее разработка
- Использует Django механизмы
- Проще поддержка
- Меньше зависимостей
