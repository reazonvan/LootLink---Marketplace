## 🎯 Следующие шаги

### Шаг 1: Запусти Pencil
1. Найди Pencil в меню Пуск (или на рабочем столе)
2. Запусти приложение
3. File → New → выбери "Web Page"

### Шаг 2: Создай первый wireframe (Homepage)
1. Назови страницу: "Homepage - Wireframe"
2. Добавь базовые блоки:
   - Header (навигация)
   - Hero Section (заголовок + 2 кнопки + 3 статистики)
   - How It Works (4 карточки)
   - Latest Listings (сетка 4x2)
   - Footer

3. Используй простые элементы:
   - Rectangle для блоков
   - Text для заголовков
   - Button для кнопок

### Шаг 3: Сохрани проект
- File → Save As
- Сохрани в: `D:\LootLink---Marketplace\design-pencil\LootLink-Wireframes.epgz`

### Шаг 4: Экспортируй для просмотра
- File → Export Document → PNG
- Сохрани в: `D:\LootLink---Marketplace\design-exports\wireframes\homepage.png`

## 🎨 Дизайн-система (используй в Pencil)

### Цвета:
```
Primary:    #2563eb (синий)
Success:    #10b981 (зеленый)
Warning:    #f59e0b (оранжевый)
Danger:     #ef4444 (красный)
Text:       #1e293b (темно-серый)
Background: #f8fafc (светло-серый)
White:      #ffffff
```

### Иконки:
- Только Bootstrap Icons: https://icons.getbootstrap.com/
- Размеры: 16px, 20px, 24px, 32px
- Стиль: outline или fill

### Типографика:
- Заголовки: 24-32px, weight 600-700
- Текст: 14-16px, weight 400-500
- Мелкий текст: 12px

## 🚀 Быстрый старт

### Для создания Homepage wireframe в Pencil:

1. **Запусти Pencil**
2. **Создай новый документ** (Web Page)
3. **Добавь элементы:**

```
┌─────────────────────────────────────┐
│ Header                              │
│ [Logo] [Menu] [Search] [User]      │
├─────────────────────────────────────┤
│ Hero Section                        │
│                                     │
│ Покупайте и продавайте игровые     │
│ предметы безопасно                  │
│                                     │
│ [Начать продавать] [Посмотреть]    │
│                                     │
│ [1234]        [567]        [890]    │
│ Пользователей Объявлений  Сделок   │
├─────────────────────────────────────┤
│ How It Works                        │
│                                     │
│ [1]      [2]      [3]      [4]     │
│ Выбор    Связь    Сделка   Отзыв   │
├─────────────────────────────────────┤
│ Latest Listings                     │
│                                     │
│ [Card] [Card] [Card] [Card]        │
│ [Card] [Card] [Card] [Card]        │
├─────────────────────────────────────┤
│ Footer                              │
│ [Links] [Social] [Copyright]       │
└─────────────────────────────────────┘
```

4. **Сохрани:** `design-pencil/LootLink-Wireframes.epgz`
5. **Экспортируй:** `design-exports/wireframes/homepage.png`

### НЕ использовать:
- Смайлики (звезды, огонь, драгоценности, галочки)

- Bootstrap Icons (bi-star, bi-fire, bi-gem)
- Цвета из дизайн-системы
- Минимализм
- Функциональность
