## Использование

### С Claude API:
```python
import anthropic

client = anthropic.Anthropic(api_key="your-key")

# Выбрать промпт из библиотеки
prompt = "[PASTE PROMPT FROM ABOVE]"

response = client.messages.create(
    model="claude-opus-4-20250514",
    max_tokens=4096,
    messages=[{"role": "user", "content": prompt}]
)
```

### С Cline Extension:
1. Открыть Cline панель в VS Code
2. Скопировать промпт из библиотеки
3. Заполнить [PLACEHOLDERS]
4. Отправить
5. Подтвердить действия

## Советы

1. **Будь конкретным**: Заполняй все [PLACEHOLDERS] конкретными значениями
2. **Добавляй контекст**: Вставляй релевантный код, логи, ошибки
3. **Указывай ограничения**: Версии библиотек, требования проекта
4. **Проси примеры**: "Покажи пример использования"
5. **Итеративно**: Начни с простого, потом уточняй
