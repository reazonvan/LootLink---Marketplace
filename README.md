# LootLink Marketplace

LootLink Marketplace — P2P-маркетплейс для торговли игровыми предметами. Проект включает публичную часть (каталог, карточки объявлений, поиск/фильтры), личный кабинет, чат между пользователями, сделки, систему отзывов/рейтинга и административные инструменты для модерации.

Репозиторий демонстрирует продакшен-подход: контейнеризация (Docker), разделение окружений/зависимостей, CI через GitHub Actions, тесты, инфраструктурные конфиги (Nginx, systemd) и документация.

## Функциональность

### Маркетплейс и объявления
- Создание, редактирование и управление объявлениями.
- Каталог с фильтрацией и сортировкой.
- Поиск (в том числе полнотекстовый, зависит от конфигурации БД/индексов).
- Загрузка изображений и работа со статикой (подготовлено для интеграции с внешним хранилищем).

### Пользователи и профили
- Регистрация и аутентификация.
- Профиль пользователя и базовые настройки.
- Верификация email (включается конфигурацией почты).

### Сделки, отзывы, репутация
- Сценарий сделки между продавцом и покупателем.
- Отзывы и рейтинг пользователей (как элемент доверия).
- История действий/операций (зависит от включённых страниц/разделов).

### Коммуникации
- Чат между пользователями: диалоги и сообщения.
- Хранение истории переписки.
- Уведомления (возможны синхронные/асинхронные сценарии, зависит от настроек Celery).

### Администрирование
- Админ-инструменты для модерации контента.
- Управление пользователями, объявлениями, жалобами (набор сущностей зависит от текущей реализации).

## Технологии

- Backend: Python, Django
- Database: PostgreSQL
- Cache/Queue (опционально): Redis, Celery
- Web server: Nginx
- App server: Gunicorn
- Containerization: Docker, docker-compose
- Quality: pytest, coverage (настроечные файлы в репозитории)
- CI: GitHub Actions (`.github/workflows/`)
